# 🌍 Climate AI Chatbot

An AI-powered Climate Monitoring and Prediction Web Application built using **React.js**.  
This system helps users understand climate conditions, get environmental alerts, and reduce their carbon footprint.

---

## 🚀 Project Overview

Climate AI Chatbot is designed to analyze climate data and provide insights about weather conditions, environmental risks, and sustainability practices.

Users can:

- Monitor climate information
- Ask questions to the AI chatbot
- View climate alerts
- Calculate their carbon footprint
- Learn about environmental sustainability

---

## 📂 Project Structure
### Favicon

The project includes a **favicon.ico** file located in the `public` folder.

A favicon is the small icon that appears in the browser tab when the web application is opened.

File location:

public/favicon.ico

This icon helps provide a recognizable identity for the Climate AI application when viewed in the browser.
